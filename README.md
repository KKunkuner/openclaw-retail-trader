# 散户.skill

> *不是分析散户，而是直接变成散户。*

一个面向中国 A 股场景的 `OpenClaw` skill。  
它会直接站进散户头脑，用第一视角输出交易想法，而不是站在外部做理性点评。

这个项目的目标很简单：
- 用散户脑子看盘
- 用散户口吻说话
- 用散户冲动下结论
- 最后直接落到“我会怎么干”

[安装](#安装) · [使用](#使用) · [输出风格](#输出风格) · [项目结构](#项目结构)

## 安装

把这个 skill 目录放进你的 OpenClaw / skill 目录即可。

最少保留这些文件：


## 使用

适合这类输入：

```text
按 A 股散户思路分析 000xxx
```

```text
我想看这票如果按超短散户的脑子会怎么下单
```

## 当前默认输出

这个 skill 现在默认保留 3 项输出：

- `我是什么路子`
- `我心里在想什么`
- `我会怎么干`

默认效果类似这样：

```text
我是什么路子: 龙头追涨型
我心里在想什么: 这票现在人气还在，我最怕的不是亏，是它真回封以后我没上车。虽然我也知道这位置追进去容易接最后一棒，但只要今天还能继续顶，我就会觉得后面还有人来接。真要给我一个回封动作，我大概率还是忍不住会跟。
我会怎么干: 真回封或者再往上顶一下我就跟，小仓先上，强了再说
```

---

## 输出风格

这套 skill 现在有几条核心硬规则：

- **全部第一视角**
  不写“散户会怎么想”，只写“我现在怎么想”。


- **情绪优先于理性**
  它不是投研助手，而是散户脑内独白生成器。

---

## 行情数据

当前版本的规则是：

- 股票数据**只允许使用 `eastmoney` skill** 获取（别的api会给错误数据）
- 需要到东方财富app通过妙想ai查询ai skills，然后将skill复制到openclaw对话窗口即可
- 如果 `eastmoney` 拿不到数据，再退回到用户提供的数据或场景化判断

默认优先关注这些字段：
- 最新价格
- 涨跌幅
- 成交额
- 换手率
- 分时强弱
- 板块联动
- 封板 / 炸板 / 回封状态

## 散户心理模型

这个 skill 内置了一组更偏真实散户的行为和心理：

- FOMO 追涨杀跌
- 止盈很快、止损很慢
- 频繁交易
- 过度自信
- 跟风
- 信息噪音交易
- 研究时间极短的冲动下单
- 频繁换策略
- 锚定买入价
- 抄底上瘾
- 情绪比逻辑重要
- 不断自我打脸、不断重写“底部”叙事
- 幻想式持仓
- “这波赚了我就辞职”
- 想尽快赚钱，偏爱日内 / 短线 / 超短线

本项目是一个模拟“散户交易行为”的 AI，仅用于学习和娱乐。

它的目标不是赚钱，而是：
