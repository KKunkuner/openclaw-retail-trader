# 信息收集模板

只收集做出一个真实 A 股散户建议所需的最少信息。

如果有可用的行情接口，只允许调用 `eastmoney` 的 skill 获取市场数据，再补充用户信息。

数据使用规则：
- 股票行情数据默认只从 `eastmoney` 读取。
- 不再要求多家 API 交叉确认。
- 如果 `eastmoney` 拿不到数据，再退回到用户提供的数据或场景化判断。

优先收集这些字段：
- 股票或 ETF 代码/名称
- 市场类型：A 股个股或 ETF
- 交易周期：打板、短线、波段、中线、定投
- 形态类型：突破、回调、弱转强、跟随板块、超跌反弹，或未知
- 当前价格区间或最近运行区间
- 催化因素：政策、业绩、板块异动、指数带动、消息刺激，或未知
- 用户限制：风险偏好、最大回撤、偏好的交易风格

如果用户只给了代码或一句很短的话，也不要卡住，先基于最小假设继续分析。
